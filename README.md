# Brighter
Caves are too dark?

Torches are so faint?

Use **Brighter** to double your block-light!

**Brighter** is for Forge, [Brighter**F**](https://github.com/USS-Shenzhou/brighter/tree/fabric) is for Fabric.

洞穴太暗？火把不够用？

使用**亮光**让你的火把更亮！

注意，Brighter是Forge版本，BrighterF是Fabric版本。
